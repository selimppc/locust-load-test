# Locust Load Testing Tool

### Installation

#### Step 1

    | venv |
    $ python -m venv venv

#### Step 2

    | Activate venv |
    $ source venv/bin/activate

#### Step 3

    | Install libs |
    $ pip install -r requirements.txt

#### Step 

    | Run Locust |
    $ locust -f path/locustfile.py

    | Web panel |
    http://0.0.0.0:8089
